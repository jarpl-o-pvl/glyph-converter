# 🛠️ Glyph Converter

**Convert Audacity label tracks into a ready-to-use glyph ringtone for Nothing Phone (1), (2), (2a), (3a), and (4a).**

> One command. One `.ogg` file. One custom light show on your Nothing Phone.
